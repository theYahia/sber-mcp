# /balance — Баланс по счетам

## Алгоритм

1. Вызови `get_accounts` для получения списка счетов
2. Если указан номер счёта — отфильтруй
3. Покажи баланс по каждому счёту

## Формат ответа

```
## Счета Сбербанк Бизнес

| Счёт | Валюта | Баланс |
|------|--------|--------|
| 40702...1234 | RUB | 1 250 000.00 |
| 40702...5678 | USD | 15 000.00 |

**Итого RUB**: 1 250 000.00
```

## Примеры

```
/balance
/balance 40702810123456781234
```
